Checklist presencial

Para realizar o transporte de cargas no Brasil é muito importante certificar que os itens de segurança obrigatórios do caminhão estejam em dia. É recomendado realizar o checklist de segurança sempre que este estiver programando para viagem monitor. Esta é a lista de equipamentos de segurança checados no PA - Supersonic. Os itens mencionados são obrigatórios para viagens monitoradas pela Komando. Portanto, caso o responsável pelo veículo não cumpra com esta listagem, pode ser autuado.
